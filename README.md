# machine learning projects
A collection of machine learning projects built with python and scikit learn covering 
real world classification and regression problems
tech stack:- python, numpy, scikit learn, pandas, matplotlib
# projects
### 1. Cancer Detection (`cancer.py`)
Predicts whether a tumor is malignant or benign based on medical features.

- **Type:** Binary Classification
- **Dataset:** Breast Cancer Wisconsin Dataset (built into Scikit-learn)
- **Algorithm:** Logistic Regression / SVM
- **Goal:** Early detection support using cell nucleus measurements

---

### 2. Customer Churn Prediction
Predicts whether a customer will **leave a service or stay**,
helping businesses take action before losing customers.

- **Type:** Binary Classification  
- **Dataset:** Telecom Customer Churn Dataset
- **Algorithm:** Random Forest / Logistic Regression
- **Goal:** Identify at-risk customers based on usage patterns

---

### 3. House Price Prediction (`house prediction.py`)
Predicts the **price of a house** based on features like
size, location, and number of rooms.

- **Type:** Regression
- **Dataset:** Boston Housing / Custom Dataset
- **Algorithm:** Linear Regression
- **Goal:** Estimate property value from structural and location features

---

### 4. SMS Spam Detection (`sms.py`)
Classifies an SMS message as **spam or not spam**
using natural language processing techniques.

- **Type:** Text Classification 
- **Dataset:** SMS Spam Collection Dataset (UCI / Kaggle)
- **Algorithm:** Naive Bayes
- **Goal:** Filter messages and predict whether they are spam or ham
